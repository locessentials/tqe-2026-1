5. Methodology Notes

Exact match, span F1, and Cohen's κ use the same calculations as the single-project agreement notebook (§5.1–5.3), pooled across all annotator pairs and documents in each round. Rounds with two translations pool both files; task IDs are namespaced per file so annotations of different translations are never compared against each other. κ is computed on spans overlapping ≥ 50% and weighted by pair size. Lead time is Label Studio's recorded annotation lead time.
